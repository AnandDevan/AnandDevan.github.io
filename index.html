Tile based jigsaw puzzle

Play game by rearranging the tiles to form the original image. You can even load your own image to create a custom puzzle.


Libraries, technology:
HTML5 canvas, audio, JavaScript

+ using html canvas drawImage to create tiles
+ radomize tile order to create puzzle
+ capture mouse, touch events to drag and place tiles
+ add html audio to confirm tile placement
+ use html input to load user images

Balloon Burst


Play game by touching (clicking on) balloon to burst



Libraries, technology:
box2d, HTML5 canvas, JavaScript

+ Used only canvas rendering to draw balloon
+ Used box2d physics engine for simulation, detecting and handling collisions
+ use html audio to play sound for bursting balloon

Libraries, technology:
webGL (three.js), custom physics engine, JavaScript
