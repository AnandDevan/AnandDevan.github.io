Tile based jigsaw puzzle

The aim of the project is to create an online version of classic jigsaw puzzle game. To make it simple it was decided that the puzzle will be formed from rectangular tiles. It was aimed to make this work on mobile devices and allow the user to form a puzzle from pictures on their devices or by taking a new picture.


Puzzle organization

We use HTML5 canvas element to present a puzzle picture and a solution picture. A set of about 20 images is used for the game as seed data and a random image is picked each time the game is played

Forming tiles (jigsaw pieces) for puzzle

HTML5 canvas drawImage is extermely flexible, it supports drawing a sub-region a picture on to a sub-region of the canvas. Once we determine a tile size (3x3 grid for now) we can draw out the puzzle by selecting subregions of the image and drawing it out to the puzzle canvas. For generating the puzzle we use the fisher-yates shuffle to randomize the puzzle tiles

Drag and drop tiles to play

We track the mousedown events to find out which tile has been selected. With every mousemove we draw move the tile and on mouseup we swap out the dragged tile and the tile which is under the mouse. Since the interactions are simple mapping touch events on mobile devices to the mouse event equivalent works very well

User cues for swapping tiles

While dragging a tile we draw out the target tile in orange to give the player some idea of swapping. And on mouseup we play a sound to indicate the tiles were swapped

Refine layout

The idea is to make use of as much screen space as possible regardless of image size or device. It involved sizing the puzzle and solution canvases to occupy most of the page size and using css to center them both.

Obtain image and sound assets

There are internet sites that make available images and sound assets royalty free. I use the following websites: flickr, openphoto, morguefile , freesound . I am thankful for these resources that improve the game experience

Loading user pictures

HTML5 input tag supports loading user files and on mobile devices this also make use of the camera to take a picture and create a new file
